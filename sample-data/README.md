# Sample data

Neutral, synthetic placeholder images for development and CI — **no real patient
data, ever.** These stand in for foot photos so the capture/quality/export code
can be exercised without a camera.

- `neutral-*.png` — plain synthetic images generated by `make_placeholders.py`.

Regenerate:

```bash
python3 sample-data/make_placeholders.py
```

The simulated camera source (Phase 2) uses these so tests and the dev
environment never require a physical camera or any real image.
